# E-Learning Management System

A backend-focused E-Learning Management System developed using Java and Spring Boot.  
This application provides RESTful APIs to manage Courses, Users, and related learning data.

## Features
- User management (Students)
- Course management
- RESTful API development
- CRUD operations
- Layered architecture (Controller, Service, Repository)

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- REST APIs
- Maven
- MySQL

## Project Structure
- Controller layer for handling API requests
- Model layer for entity mapping
- Repository layer for database operations
- Application properties for configuration

## How to Run the Project
1. Clone the repository
2. Configure database details in `application.properties`
3. Run the application using:
4. 4. Access APIs using Postman or browser.
